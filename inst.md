
## РАЗВЕРТЫВАНИЕ СРЕДЫ: ТВОЙ ВЕРСТАК (VS CODE + ANACONDA)
В аналитике никто не пишет код в голом Блокноте. Тебе нужен тяжелый промышленный инструмент.

   1. Скачай Anaconda Distribution (это бесплатный пакет, где уже установлен сам Python и ВСЕ аналитические библиотеки: Pandas, NumPy, SQLAlchemy).
   2. Скачай VS Code (Visual Studio Code) — твой главный текстовый редактор.
   3. Настройка внутри VS Code:
   * Зайди во вкладку Extensions (расширения, иконка четырех квадратов слева).
      * Установи расширение Python (от Microsoft).
      * Установи расширение Jupyter (от Microsoft).
   4. Создай первый боевой файл: Создай папку на компьютере, открой ее в VS Code и создай файл с расширением .ipynb (например, python_master.ipynb). Это интерактивный блокнот Jupyter. Код в нем выполняется не целиком, а отдельными ячейками (нажал Shift + Enter — выполнилась конкретная строка, результат отобразился сразу под ней). Это твой идеальный полигон для тестов.
